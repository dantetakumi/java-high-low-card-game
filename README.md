## High-Low Showdown - Java Console Card Game

- High-Low Showdown is an object-oriented console-based card game developed in Java. The player competes against a computer opponent by guessing whether the next card drawn from a shuffled deck will be higher or lower in value. The project emphasizes multi-class system design, inheritance and polymorphism, user input handling, and structured game logic.

### Features
- Interactive console-based card guessing game
- Player competes against a computer opponent
- Full 52-card deck generation and shuffling
- Round-based gameplay loop with continuous play
- Score tracking and streak bonus system
- Win, lose, tie, and quit conditions
- Input validation for incorrect user entries
- Clear and readable scoreboard display
- Final results summary upon game completion

### Technical Concepts 
- Object-Oriented Programming (OOP) in Java
- Multi-class program architecture
- Inheritance and polymorphism through Player subclasses
- Array data structures for deck management
- Loop constructs for repeated gameplay rounds
- Conditional logic for win/loss determination
- Scanner-based user input handling
- Method parameters and return values
- Encapsulation and modular code organization

### Technologies Used
- Java
- BlueJ IDE

### Project Structure
- Card System: Represents playing card rank, suit, and value
- Deck System: Manages deck creation, shuffling, and card drawing
- Player System: Stores player data including name, score, and streak
- Human Player: Handles user input and decision-making
- Computer Player: Generates automated guesses
- Game Manager: Controls round flow, rules, and game progression
- Scoreboard System: Tracks scores, streaks, and displays results
- Main Controller: Initializes and starts the game
